# MLHW8

## Environment

+ Required Package

    + Python 3.6+
    + Scikit-Learn
    + Numpy
    + Pytorch
    + torch_optimizer, installed by
        
        ```bash
        pip install torch_optimizer
        ```
## Training and Testing
1. Put csv files into the project root directory.

2. put face images at {SRCROOT}/faces

3. Run

    ```bash
    python main.py
    ```
